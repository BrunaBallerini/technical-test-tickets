# Critérios de Avaliação
Este teste avalia quatro competências fundamentais.

### 1. Manipulação de Dados Complexos

**O que avaliamos:**
- Capacidade de trabalhar com dados reais e imperfeitos
- Tratamento adequado de dados faltantes e inconsistências
- Limpeza e preparação eficiente dos dados
- Identificação e correção de anomalias
- Transformações apropriadas para diferentes tipos de variáveis

### 2. Pensamento Analítico

**O que avaliamos:**
- Capacidade de extrair insights relevantes dos dados
- Formulação de hipóteses e validação através de análise
- Identificação de padrões e tendências significativas
- Raciocínio lógico na abordagem do problema
- Conexão entre achados técnicos e contexto de negócio

### 3. Capacidade de Modelagem

**O que avaliamos:**
- Seleção apropriada de algoritmos para o problema
- Implementação técnica correta dos modelos
- Estratégias adequadas de validação e avaliação
- Feature engineering criativo e relevante
- Interpretação correta dos resultados

### 4. Comunicação de Resultados

**O que avaliamos:**
- Clareza na apresentação de metodologia e resultados
- Capacidade de traduzir achados técnicos para linguagem de negócio
- Qualidade das recomendações práticas
- Identificação honesta de limitações
